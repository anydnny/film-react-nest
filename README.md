# FILM!
Разработка веб-приложения для просмотра фильмов и бронирования билетов.

Стек: React, NestJS, TypeScript, PostgreSQL, MongoDB, TypeORM, Mongoose

Что было сделано:
- реализован frontend на React и backend API на NestJS
- разработаны REST-эндпоинты, контроллеры и сервисы для управления фильмами и сеансами
- реализована бизнес-логика бронирования билетов и обновления доступных мест
- реализовано хранение данных через MongoDB (Mongoose) и PostgreSQL (TypeORM)
- реализован конфигурируемый backend с выбором СУБД через переменные окружения
